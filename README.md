## Instalação

    - Clonar repositorio
    $ git clone https://github.com/jveiiga/project-counter.git

    FRONT-END 📺
    
    $ cd project-counter

    - Instalação de dependências
    $ yarn install
    ou
    $ npm install

    - Rodar projeto
    $ yarn dev
    ou
    $ npm dev
